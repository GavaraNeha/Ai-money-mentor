# AI Money Mentor 💰

AI Money Mentor is a financial planning system designed for students to analyze income, expenses, and savings.

## Features
- Money Health Analysis
- Expense Breakdown (Pie Chart)
- Savings Calculation
- Financial Advice

## Tech Stack
- Python (Flask)
- HTML, CSS
- Chart.js

## How to Run
1. Clone repo
2. Install requirements
3. Run app.py
